Vscode 写Golang 的小技巧

首先装好Vscode 编辑器 插件 VsCode-Go. 一切安装正常的。

1.定义好结构体，快速生成tag.

2. 先把光标定位到结构体那行，再按 Mac 快捷键 Cmd + Shift + P 调出命令行。

3. 选择 Go: Add Tags To Struct Fields 命令, 然后回车。 默认生成的 tag: json

自定义tag 标签

修改配置文件 settings.json

重复上面的步骤 1,2,3 。 搞定。
